사용방법

byteexam 1s — AI로 시험지를 만드는 방법을 안내합니다.

1

AI 프롬프트 생성기에서 프롬프트 복사

캔버스 중앙 또는 상단 툴바의 「✨ AI 프롬프트」 버튼을 클릭합니다.

  • 시험 범위를 입력합니다. (예: 이차함수, 확률과 통계 — 순열과 조합)
  • 교육과정 / 학년 / 난이도 / 문항 수를 선택합니다.
  • 필요하면 추가 요청을 적습니다. (예: 서술형 2문제 포함)
  • 「프롬프트 복사」 버튼을 누릅니다.
2

AI에 붙여넣기 → 코드 생성

복사한 프롬프트를 Claude / GPT / Gemini 등 AI 채팅에 붙여넣습니다.

AI가 아래 형식의 코드를 출력합니다:

---BYTEEXAM-START---
[HEADER]
title=이차함수 단원 평가
subject=이차함수
difficulty=실력
date=2026-05-11
total=20

[QUESTION]
num=1
type=5choice
text=이차함수 $y=x^2-4x+3$의 꼭짓점의 좌표는?
choice1=① $(1, 0)$
choice2=② $(2, -1)$
choice3=③ $(3, 0)$
choice4=④ $(-2, -1)$
choice5=⑤ $(2, 1)$
answer=2
point=4
explain=완전제곱식으로 변형: $y=(x-2)^2-1$, 꼭짓점 $(2,-1)$

[ANSWER_SHEET]
1=②
...
---BYTEEXAM-END---
3

코드 붙여넣기 → 시험지 등록

  • 왼쪽 패널 상단 「+ 새 시험지」 버튼을 누릅니다.
  • AI가 준 코드를 내용 입력란에 그대로 붙여넣습니다.
  • 제목을 확인하고 「저장하기」를 누릅니다.
  • 왼쪽 패널 목록에서 방금 만든 시험지를 클릭하면 캔버스에 렌더링됩니다.
  • 문제를 클릭하면 오른쪽 패널에서 바로 편집할 수 있습니다.
4

PDF 저장 / 인쇄

  • 「PDF 다운로드」 — 페이지가 자동으로 A4 2단 레이아웃 PDF로 저장됩니다.
  • 「인쇄」 — 브라우저 인쇄 창에서 「PDF로 저장」을 선택하면 고화질 출력이 가능합니다.
  • 문제가 페이지 경계선에 걸치지 않도록 자동으로 배치됩니다.

📋 코드 필드 참고

필드 설명 예시
type=문제 유형5choice / 5choice_with_bogi / 5choice_with_short / bogi_with_short / bogi_with_essay / short / essay
text=문제 본문수식은 $…$ 또는 $$…$$
bogi_1~3=보기 항목 (ㄱ,ㄴ,ㄷ)ㄱ. 명제 내용
choice1~5=선택지① 내용 (원문자 필수)
answer=정답 번호 (5지선다)3 (choice3이 정답)
answer_text=정답 텍스트 (단답·서술형)x=2 또는 모범 답안
point=배점2 / 3 / 4 / 5
explain=해설단계별 풀이 + 오답 원인

💡 팁

수식 입력 — LaTeX 문법을 사용합니다. 인라인은 $x^2+1$, 블록은 $$\frac{a}{b}$$
보기형 문제 — type=5choice_with_bogi 사용 시 bogi_1, bogi_2, bogi_3 필드를 추가합니다.
화학 구조식 — [SMILES:c1ccccc1] 형식으로 입력하면 구조식이 자동으로 렌더링됩니다.
AI 재생성 — 마음에 들지 않는 문제가 있으면 AI에게 "3번 문제를 더 어렵게 바꿔줘" 처럼 요청한 후 코드를 다시 붙여넣고 저장하면 됩니다.
